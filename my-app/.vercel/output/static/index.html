Noahh is a fan-made extension for Geometry Dash that adds mod support to the game. Browse from an in-game list to seamlessly download mods on Windows, Mac, Android, and iOS!

Noahh is the most popular GD mod loader across all platforms. With an active community of both users and modders, nearly every mod you can imagine has been made or suggested!

Noahh is open-source and is designed to make the modding experience infinitely smoother for developers. Noahh comes with a special hooking syntax as well as dozens of built-in UI components, utility functions, and everything else needed to make mods.

Documentation Examples 
// Include Noahh headers
#include <Noahh/modify/MenuLayer.hpp>;

using namespace noahh::prelude;

// Add hooks to the main menu
class $modify(MenuLayer) {
    // Override the main menu creation
    bool init() {
        // Start off by creating the original main menu
        if (!MenuLayer::init())
            return false;
        
        // We are now modding!
        log::info("Hi from my mod!");

        return true;
    }
};
